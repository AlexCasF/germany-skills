# Python implementation

Run with:

```powershell
python skills\bundestag-lobbyregister\python\lobbyregisterctl.py doctor
```

The Python version mirrors the 2.0 command contract used by the Go binary:
`doctor`, `statistics`, `search`, `entry get`, `entry source`,
`entry dossier`, `financial summary`, `statements list`, and `v1 search`.

Set `LOBBYREGISTER_API_KEY` before live V2 calls.
