# Bundeshaushalt Skill

Bundeshaushalt Digital CLI skill for German federal budget research.

Primary runtime:

```powershell
skills\bundeshaushalt\bin\bundeshaushaltctl-2.0.exe doctor
```

Alternative runtimes:

```powershell
python skills\bundeshaushalt\python\bundeshaushaltctl.py doctor
node skills\bundeshaushalt\typescript\dist\index.js doctor
```

Start with `SKILL.md`, then use `references/notes.md` and `tests/test-plan.md` for implementation details.
