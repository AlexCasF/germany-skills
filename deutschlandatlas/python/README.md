# Deutschlandatlas Python CLI

Python parity implementation for `deutschlandatlasctl` using only the standard
library.

## Run

```powershell
python skills\deutschlandatlas\python\deutschlandatlasctl.py doctor
python skills\deutschlandatlas\python\deutschlandatlasctl.py table sample --table alq_HA2023 --limit 2
```

The command surface mirrors the Go 2.0 implementation.
