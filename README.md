# Germany Skills

Skill-guided CLI tools for German public-data research.

This repository contains the API-family skill directories that were split out of
`AlexCasF/democracy-researcher`.

Each skill directory is self-contained and may include:

- `SKILL.md` usage guidance for agents
- API notes and OpenAPI/reference files
- Go, Python, and TypeScript CLI implementations
- compiled CLI binaries where already produced

The companion agent application lives in `democracy-researcher`.
