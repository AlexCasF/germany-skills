# Abgeordnetenwatch documentation reference

Official documentation page:

- `https://www.abgeordnetenwatch.de/api`

Documented endpoint families observed on the page and confirmed via the live API:

- `/api/v2/parliaments`
- `/api/v2/parliament-periods`
- `/api/v2/politicians`
- `/api/v2/candidacies-mandates`
- `/api/v2/polls`

The documentation also explains:

- filter operators
- nested filtering via referenced entities
- pagination and range limiting
- sorting
- `related_data` expansions
